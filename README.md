# flask-graphene-hello
A simple example using GraphQL (Graphene) and Flask


### Prerequisites

```
$ pip install pipenv
```

### Installing

1. Install dependencies
```
$ pipenv install
```

2. Open project shell

```
pipenv shell
```


3. Run Flask
```
FLASK_APP=./server/index.py flask run
```

Go to `localhost:5000/graphql`
